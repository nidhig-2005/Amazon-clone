Amazon Clone – HTML & CSS

A simple and responsive Amazon homepage clone built using only HTML and CSS.
This project recreates the layout, design, and UI components of the Amazon India website without using JavaScript or backend functionality.

Features:
Fully recreated Amazon-style homepage UI
Responsive layout using pure CSS
Hero section banner
Product cards with hover effects
Navigation bar similar to Amazon
Footer with multi-section links
Clean and professional code structure

Technologies Used:
HTML– structure
CSS – styling, layout, responsiveness
(No JavaScript used)

Purpose of This Project:
This project was built to:

Practice HTML & CSS
Understand real-world web layout
Improve UI design skills
Strengthen frontend fundamentals
